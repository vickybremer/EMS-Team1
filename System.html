WORTSCHATZ

DAS SYSTEM

Wortschatz im inaktiven Zustand. Die Box ist im Standby Modus und leuchtet auf, wenn neue Vokabeln bei ihr ankommen.

Mit dem Projekt Wortschatz soll der Merkprozess beim Lernen von Englischvokabeln für Schüler:innen optimiert werden. Wortschatz greift das bekannte System einer Karteikartenbox auf. Sie wird durch Technologie unterstützt und fördert insbesondere das Erstellen von Assoziationen zu den Lerninhalten sowie das regelmäßige Wiederholen der Vokabeln anhand von Spielen.

Die Lehrperson teilt die Vokabeln einer Unit zu Beginn dieser in kleine Blöcke auf. Diese werden über einen socket.io Server an die verschiedenen Vokabelboxen (Clients) gesendet, wo sie auf die einzelnen Karten beschrieben werden. So erhält jede Karte innerhalb der Box eine neue Vokabel. Jeden Tag der Schulwoche steht eine kleine Aufgabe an, die anhand eines Spieles zu bearbeiten ist.

Mit einem Klick auf die Knöpfe können Sie sehen wie Wortschatz funktioniert!
